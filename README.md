# Height or Weight Prediction using Linear Regression

## Overview
This project builds a predictive machine learning model to estimate the height or weight of a person using a dataset containing information about individuals' heights and weights.

## Dataset
SOCR Height and Weight Dataset

Dataset Link:  
http://wiki.stat.ucla.edu/socr/index.php/SOCR_Data_Dinov_020108_HeightsWeights

The dataset contains measurements of human height and weight which are used to train a regression model.

## Objective
To develop a machine learning model that learns the relationship between height and weight and predicts one variable based on the other.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Algorithm Used
Linear Regression

## Steps Involved
1. Import required libraries
2. Load the dataset
3. Perform data preprocessing
4. Visualize the relationship between height and weight
5. Split the dataset into training and testing sets
6. Train the Linear Regression model
7. Make predictions using the trained model
8. Evaluate the model performance

## Result
The trained regression model successfully predicts height or weight based on the relationship learned from the dataset.

## Conclusion
Linear Regression helps understand the relationship between height and weight and can be used to predict one variable when the other is given.
